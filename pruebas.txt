Ruta http://localhost:3000

Al ingresar a esta ruta el servidor responde con el mensaje “Servidor activo”. Esto indica que el servidor está funcionando correctamente.

Ruta http://localhost:3000/saludo

Al ingresar a esta ruta el servidor muestra el mensaje “Hola comunidad”. La ruta sirve para enviar un saludo desde el servidor.

Ruta http://localhost:3000/mensaje/juan

Al ingresar a esta ruta el servidor responde con el mensaje “Hola juan”. En esta ruta se utiliza un parámetro dinámico para mostrar el nombre escrito en la URL.
